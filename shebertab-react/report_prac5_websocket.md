# №5 Практикалық жұмыс есебі

**Тақырыбы:** Нақты уақыт хабарламалары мен хабарландырулар жүйесін (WebSocket / Socket.io) іске асыру.
**Жоба:** SheberTab — мамандарды табу платформасы.

---

## 1. Жұмыстың мақсаты
Платформадағы пайдаланушылар (клиенттер мен мамандар) арасында нақты уақыт режимінде (real-time) ақпарат алмасуды қамтамасыз ету. Бұған чат хабарламалары, тапсырыс күйінің өзгеруі туралы хабарландырулар және интерфейстің бетті жаңартпай-ақ автоматты түрде жаңаруы жатады.

## 2. Қолданылған технологиялар
- **Backend:** Node.js, Express, Socket.io
- **Frontend:** React.js, Socket.io-client
- **Дерекқор:** PostgreSQL (хабарламалар тарихын сақтау үшін)

---

## 3. Backend бөлігінде іске асыру

Серверде `Socket.io` кітапханасы арқылы нақты уақыт режиміндегі байланыс орнатылды. Негізгі логика `server.js` файлында орналасқан:

### 3.1. Socket серверін баптау:
```javascript
const http = require('http');
const { Server } = require('socket.io');

const server = http.createServer(app);
const io = new Server(server, {
  cors: {
    origin: "http://localhost:5173", // Frontend адресі
    methods: ["GET", "POST"]
  }
});
```

### 3.2. Пайдаланушыларды бөлмелерге (Rooms) біріктіру:
Әрбір пайдаланушы жүйеге кіргенде өзінің `userId`-і бойынша жеке бөлмеге қосылады. Бұл хабарламаларды нақты бір адамға бағыттап жіберуге мүмкіндік береді.
```javascript
io.on('connection', (socket) => {
  socket.on('join', (userId) => {
    socket.join(`user_${userId}`); // Пайдаланушыны жеке бөлмесіне қосу
    console.log(`User ${userId} joined their room`);
  });
});
```

### 3.3. Route-тарда Socket-ті қолдану:
Жаңа хабарлама немесе тапсырыс туралы хабарландыру жасалғанда, сервер автоматты түрде Socket арқылы эвент жібереді:
```javascript
app.use((req, res, next) => {
  req.io = io; // Socket-ті барлық маршруттарға қолжетімді ету
  next();
});

// Мысалы, хабарлама жіберу кезінде:
req.io.to(`user_${receiverId}`).emit('new_message', newMessage);
```

## 4. Frontend бөлігінде іске асыру

Frontend жағында `Navbar.jsx` және `Messages.jsx` компоненттерінде сервермен байланыс орнатылды.

### 4.1. Серверге қосылу:
```javascript
import io from 'socket.io-client';
const socket = io('http://localhost:5000');

useEffect(() => {
  if (user) {
    socket.emit('join', user.id); // Серверге өзімізді таныстыру
  }
}, [user]);
```

### 4.2. Нақты уақытта хабарлама алу:
Хабарламалар келіп түскенде интерфейс лезде жаңарады:
```javascript
useEffect(() => {
  socket.on('new_message', (msg) => {
    setMessages(prev => [...prev, msg]); // Чат тізімін жаңарту
    // Хабарландыру баджын немесе дыбысын көрсету
  });
  
  return () => socket.off('new_message');
}, []);
```

---

## 5. Жұмыс нәтижесі

1.  **Жедел чат:** Пайдаланушылар бір-біріне хабарлама жазғанда, ол екінші тарапқа бетті жаңартпай-ақ бірден көрінеді.
2.  **Интерактивті хабарландырулар:** Тапсырыс қабылданғанда немесе аяқталғанда, пайдаланушының навигациялық панелінде (Navbar) көк нүкте (unread indicator) пайда болады.
3.  **Анимациялар:** Хабарландырулар тізімі ашылғанда `dropdownFadeIn` анимациясы арқылы сұлу көрінеді.

## 6. Қорытынды
Бұл практикалық жұмыста WebSocket технологиясын қолдану арқылы SheberTab платформасының интерактивтілігі артты. Socket.io қолдану пайдаланушылардың платформадағы жұмыс тиімділігін арттырып, хабарламаларды күту уақытын нөлге дейін азайтты.
