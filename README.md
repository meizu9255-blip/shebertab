## 📁 Жоба құрылымы

```
online-test-system/
├── css/
│   └── style.css          ← Барлық стиль (Cyan-Teal дизайн)
├── js/
│   ├── practical2.js      ← Тапсырма 1–6 JavaScript логикасы
│   └── auth.js            ← Тапсырма 7–9 + Lamp анимациясы
├── index.html             ← Басты бет
├── about.html             ← Біз туралы
├── services.html          ← Қызметтер + баға тізімі
├── blog.html              ← Блог мақалалары
├── contact.html           ← Байланыс формасы
├── practical2.html        ← JS Зертхана (1–6 тапсырма)
├── register.html          ← Тіркелу (7-тапсырма)
└── login.html             ← Кіру (8–9 тапсырма)
```

## ✅ Орындалған тапсырмалар

### `practical2.html` — JavaScript Зертхана
| # | Тапсырма | Технология |
|---|----------|-----------|
| 1 | Батырма арқылы мәтін/фон өзгерту | `addEventListener`, `style.display` |
| 2 | Форма валидациясы | `value`, `regex`, `classList` |
| 3 | DOM — элемент қосу/жою | `createElement`, `appendChild`, `remove` |
| 4 | Блок көрсету/жасыру | `classList.toggle`, CSS `max-height` |
| 5 | Секунд санауыш | `setInterval`, `clearInterval` |
| 6 | Стиль өзгерту | `classList.add/remove/toggle` |

### `register.html` + `login.html` — Auth беттері
| # | Тапсырма | Технология |
|---|----------|-----------|
| 7 | Тіркелу + пароль күші | `localStorage`, `JSON.stringify` |
| 8 | Кіру + деректерді тексеру | `array.find()`, `JSON.parse` |
| 9 | Auth state + Logout | `localStorage`, DOM manipulation |

**Бонус:** Lamp Pull анимациясы — физикалық серіппе + тербеліс эффекті
